## 💻 Running the Project Locally
To run the portfolio site locally, simply clone the repository and start a local HTTP server:
```bash
# Clone the repository
git clone https://github.com/Rahul006-max/portfoliov2.git

# Navigate into the project
cd portfoliov2

# Start a simple Python server to host index.html
python -m http.server 8000
```
Open **`http://localhost:8000`** in your browser.
