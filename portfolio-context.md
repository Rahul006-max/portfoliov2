## 4. Strict Design Rules & Reference

- **Design Reference Website:** https://hey.milo.gg/ (Use this site as the absolute baseline for the UI layout and interaction physics).
- **Aesthetic:** Interactive Scrapbook / Neo-Brutalism. Floating, draggable cards on a stark background. High contrast, monochrome (black and white) to reflect data precision.
- **Iconography:** Use ONLY minimalist geometric icons (e.g., Lucide React). **Strict rule: ABSOLUTELY NO EMOJIS.**
- **Typography & UI:** Proper text alignment is mandatory. Focus strictly on technical metrics and data outcomes.
- **Interactivity:** Elements must feel tactile like physical cards. Solid drop-shadows (e.g., 6px solid black) that compress to 2px when clicked or dragged.
